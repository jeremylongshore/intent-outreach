# Support

GitHub issues are disabled on this repository.

- **Questions, bugs, feature requests:** email `jeremy@intentsolutions.io`.
- **Security issues:** follow `SECURITY.md` (email `security@intentsolutions.io`) — do not post publicly.
- **Commercial / enterprise licensing:** `jeremy@intentsolutions.io` · https://intentsolutions.io

Before reaching out, check `README.md` (setup + keys) and `CLAUDE.md` (architecture). Most "it didn't run"
issues are an unset provider key — `node dist/cli.js connectors` shows which connectors are configured.
